# Interactive Globe

An interactive 3D globe built with **globe.gl** (Three.js wrapper) that allows users to explore and learn about countries worldwide.

## Features

- **3D Globe Visualization**: Realistic 3D globe with Earth texture and topography
- **Hex Bin Layer**: Countries visualized as colored hexagonal clusters
- **Country Labels**: Three-letter country codes displayed on the globe
- **Interactive Exploration**: 
  - Drag to rotate the globe
  - Scroll to zoom in/out
  - Hover over hex bins or labels to interact
  - Click to view detailed country information
  - Auto-rotating globe that pauses on user interaction
- **Information Panel**: Displays country details including capital, population, area, and currency
- **Smooth Animations**: Camera transitions and smooth interactions

## Technologies

- **globe.gl** (v2.45.0): High-level globe visualization library
- **Three.js**: Underlying 3D graphics engine
- **HTML5 & CSS3**: Structure and styling
- **JavaScript**: Interactivity
